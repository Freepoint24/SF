Все тесты в одном файле не запустились.
Разбил тесты для каждого алгоритма.

sort - все одним файлом
sort1bubbleSort     - bubbleSort тесты
sort2selectionSort  - selectionSort тесты
sort3insertionSort  - insertionSort тесты
sort4mergeSort      - mergeSort тесты
sort5QuickSort      - QuickSort
sortv4 - дополнительные тесты на больших данных
Итоги.ods   - сводная таблица тестов

С малыми объемама данных справляются все алгоритмы.

bubbleSort и selectionSort - использовать только в учебных целях на малых объемах данных
insertionSort - справляется с малыми и средними объемами данных, и очень плохо справляется с большими объемами
mergeSort  и quickSort универсальные. Хорошо показали себя на больших и малых объемах.
